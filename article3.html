SQL DSL in Python

Published on June 24, 2025

Domain-Specific Languages (DSLs) simplify specific tasks through high-level syntax. I built a DSL in Python to generate SQL queries with a fluent API. It supports SELECT, JOIN, INSERT, DELETE, UPDATE, WHERE, ORDER BY, and more.

Why a DSL for SQL?

Hardcoding SQL strings can be messy and error-prone. My DSL improves code readability, type safety, and query composition. It's especially useful for dynamically generating queries based on UI inputs.

Features

The DSL supports multiple SQL dialects like SQLite, MySQL, and PostgreSQL, and can output prettified SQL. It also includes a GUI for live testing and CSV export of query results.

Key Tools Used: Python, OOP, SQLAlchemy Concepts, Tkinter
