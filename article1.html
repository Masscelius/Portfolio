How I Built My Portfolio Website

Published on June 24, 2025

As a student of both Civil and Computer Science Engineering, I wanted a personal site to showcase my unique mix of technical and creative skills. I began by sketching the layout and identifying key sections like projects, contact, and resume. Then, I coded the site using HTML, CSS, and JavaScript and hosted it on GitHub Pages for free.

Design Inspiration

I followed a modern clean aesthetic with soft shadows, smooth transitions, and accent colors like teal for personality. Animations were added using AOS and basic JavaScript to enhance the interactivity without overloading the page.

Deployment

Using Git and GitHub made collaboration and updates easy. GitHub Pages allows free static hosting, and I connected my custom domain later. I also integrated a dark mode toggle and smooth scroll features to improve user experience.

Key Tools Used: HTML5, CSS3, JavaScript, AOS, GitHub Pages
